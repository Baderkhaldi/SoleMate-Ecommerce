# SoleMate - E-Commerce Store

A dynamic full-stack e-commerce web application built using Node.js, Express.js, and a MySQL database. This project demonstrates end-to-end web development skills, from responsive front-end design to secure back-end routing.

## 🚀 Key Features

*   **Custom Product Catalog:** Designed interactive product pages and carousels using EJS, CSS, and Bootstrap.
*   **Personalized Branding:** Customized the site header and implemented dynamic product pricing adjustments to reflect a unique brand identity.
*   **Dynamic Shopping Cart:** Engineered the core routing logic for the shopping cart, handling item updates, quantity management, and subtotal calculations seamlessly.
*   **Secure Database Interactions:** Utilized parameterised SQL queries to ensure safe data handling and prevent SQL injection.

## 🛠️ Tech Stack

*   **Front-end:** HTML, CSS, Bootstrap, EJS
*   **Back-end:** Node.js, Express.js
*   **Database:** MySQL
